Frontend

¿Qué es?


Frontend, también conocido como desarrollo frontend o desarrollo del lado del cliente, se refiere a la parte de un sitio web o una aplicación web que los usuarios ven y con la que interactúan directamente. Es la capa visible y accesible de una aplicación web. El desarrollo frontend implica la creación de la interfaz de usuario (UI) y la experiencia del usuario (UX) que permite a los usuarios interactuar con el sitio de manera efectiva y atractiva.
